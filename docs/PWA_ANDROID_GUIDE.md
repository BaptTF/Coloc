### Install Issues

#### "Install" button not appearing
**Solutions:**
- **Check browser:** Use Chrome or Edge (not Firefox)
- **Update browser:** Ensure latest version
- **Clear cache:** Settings â†’ Chrome â†’ Storage â†’ Clear cache
- **Restart browser:** Force close and reopen
- **Check connection:** Ensure stable internet

#### Installation fails
**Solutions:**
- **Free up storage:** Need at least 50MB available
- **Restart device:** Clear temporary issues
- **Check Android version:** Must be 8.0+
- **Try manual method:** Use menu installation

#### App not on home screen
**Solutions:**
- **Check app drawer:** Swipe up to see all apps
- **Search for "Coloc":** Use device search
- **Check widgets:** May be in widget section
- **Reinstall:** Remove and try again

### Share Target Issues

#### Coloc not in share menu
**Solutions:**
- **Install app first:** Share target only works after installation
- **Restart source app:** Force close YouTube/Twitch
- **Clear app cache:** Settings â†’ Apps â†’ YouTube â†’ Storage â†’ Clear cache
- **Check permissions:** Settings â†’ Apps â†’ Coloc â†’ Permissions

#### Shared URL not working
**Solutions:**
- **Check URL format:** Ensure valid video URL
- **Test internet:** Ensure online connectivity
- **Try manual paste:** Copy URL manually into app
- **Report issue:** With URL details

### Performance Issues

#### App loading slowly
**Solutions:**
- **Check internet:** First load needs connection
- **Clear cache:** Settings â†’ Chrome â†’ Storage â†’ Clear cache
- **Close other apps:** Free up memory
- **Restart device:** Clear system issues

#### Downloads not starting
**Solutions:**
- **Check server status:** Ensure backend is running
- **Verify URL:** Test with different video
- **Check VLC config:** Ensure VLC is accessible
- **Review error logs:** Check download progress
